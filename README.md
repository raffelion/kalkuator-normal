# Kalkuator Normal

Project HTML, CSS, dan JavaScript buat kalkulator absurd yang pura-pura normal, padahal hasil hitungannya selalu error dan nyuruh user hubungi customer service.

## Fitur

- Tampilan kalkulator dengan animasi dan efek glow
- Loading palsu sebelum hasil selalu gagal
- Popup error yang nyuruh user hubungi CS
- Chat CS dengan indikator lagi mengetik
- Balasan spesial pertama, balasan lanjutan absurd, dan kirim stiker otomatis
- Konfeti di balasan CS pertama

## File Utama

- `index.html`
- `style.css`
- `script.js`
- `gokil-stiker.jpg`

## Cara Jalanin

1. Download atau clone repo ini.
2. Buka file `index.html` di browser.
3. Pencet tombol kalkulator sesukamu.
4. Nikmati error yang sangat tidak membantu itu.

## Catatan

Ini project iseng dan sengaja dibuat chaotic. Jadi kalau kelihatannya normal tapi ternyata ngeselin, ya memang itu tujuannya, bang 😹
